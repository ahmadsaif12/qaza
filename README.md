## 🌟 Our Mission

**Qaza Tracker** was built with a singular, sincere intention: to help Muslims worldwide manage, track, and complete their missed (Qaza) prayers. 

In the busyness of modern life, it can be overwhelming to calculate and remember how many prayers are owed over the years. Our mission is to remove that friction, providing a beautiful, intuitive, and spiritually uplifting tool that encourages consistency and accountability in fulfilling our most fundamental obligation to Allah (SWT).

*“Verily, the prayer is enjoined on the believers at fixed hours.”* — Quran (4:103)

---

## ✨ Features

- **📊 Comprehensive Tracking:** Easily track Qaza prayers for Fajr, Dhuhr, Asr, Maghrib, Isha, and Witr.
- **📈 Progress Visualization:** See your journey over time. Watch your missed prayer count go down as your dedication goes up.
- **📱 Responsive & Fast:** Built on modern web technologies ensuring a smooth experience on both desktop and mobile.
- **🌙 Beautiful UI:** A clean, minimal, and calming interface designed to help you focus on your spiritual goals without distraction.
- **🔒 Privacy First:** Your spiritual journey is personal. Data is handled securely and privately.

## 🛠️ Getting Started (For Developers)

This project is built with [Next.js](https://nextjs.org).

### Prerequisites
- **Node.js** (v18+)
- **npm** / **yarn** / **pnpm**

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/moidrk/qaza.git
   cd QazaTrack
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up Environment Variables**
   Copy `.env.example` to `.env.local` and configure it.
   ```bash
   cp .env.example .env.local
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📚 Islamic Knowledge on Qaza Prayers

### What is a Qaza Prayer?
A Qaza prayer is an obligatory (Fard) prayer that was not performed in its prescribed time. It remains an obligation until it is offered.

### How to Calculate?
If you have missed prayers for months or years, you should make an honest, conservative estimate of how many prayers you have missed since reaching the age of puberty, and begin making them up steadily. 

*May Allah make it easy for all of us to establish our prayers and accept our efforts. Ameen.*
